# 🍽️ Rodízio MJ

Sistema web desenvolvido para gerenciamento de pedidos de uma pastelaria/rodízio, permitindo que clientes registrem pedidos por mesa e que a equipe da cozinha acompanhe e finalize os pedidos em tempo real.

---

## 📖 Sobre o Projeto

O **Rodízio MJ** foi criado com o objetivo de informatizar o fluxo de pedidos de uma pastelaria, tornando o atendimento mais rápido e organizado.

O sistema é dividido em duas áreas:

### 👨‍🍳 Área da Cozinha

* Login administrativo.
* Visualização dos pedidos recebidos.
* Identificação da mesa solicitante.
* Finalização dos pedidos.
* Controle de acesso para funcionários.

### 🛒 Área de Pedidos

* Seleção de produtos.
* Adição e remoção de itens.
* Envio de pedidos por mesa.
* Armazenamento local dos pedidos.

---

## 🚀 Funcionalidades

### Cliente

✅ Realizar pedidos por mesa

✅ Adicionar múltiplos itens

✅ Remover itens antes do envio

✅ Enviar pedido para a cozinha

### Cozinha

✅ Login administrativo

✅ Visualizar pedidos pendentes

✅ Marcar pedidos como concluídos

✅ Atualização automática da lista de pedidos

✅ Logout seguro

### Sistema

✅ Persistência de dados utilizando LocalStorage

✅ Interface responsiva

✅ Design personalizado com identidade visual própria

✅ Recuperação de senha
